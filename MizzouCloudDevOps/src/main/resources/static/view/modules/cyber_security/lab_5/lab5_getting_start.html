Purpose of the Lab

In this lab, you will learn about VMware Tanzu Community Edition (TCE), a full-featured, easy-to-manage Kubernetes platform for learners and users, especially those working in small-scale or preproduction environments. It is a freely available, community supported, open source distribution of VMware Tanzu that can be installed and configured in minutes on your local workstation or your favorite cloud.

Goals/Outcomes
  • Create Tanzu cluster and deploy applications
  • Get familiar with the basic functionalities of VMWare Tanzu Community Edition.
  • Deploy Drone Video Processing Application
