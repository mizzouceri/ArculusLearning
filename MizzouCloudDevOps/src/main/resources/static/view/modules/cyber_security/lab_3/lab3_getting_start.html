Purpose of the Lab

In this lab, we will learn how to setup K3S capability on a public cloud, AWS. We will create a K3S cluster that has a master node, and a few edge (worker) nodes with different computing power to mimic different edge devices in a real world scenario. Once the cluster is setup, we will leverage a use case application - Volunteer Edge Computing with Trusted bioinformatics, and show how we can leverage the K3S cluster to orchestrate HPC workflows on selected edge nodes in the cluster.

Goals/Outcomes
  • Deploy a K3S cluster on AWS using EC2 instances.
  • Manage cluster to include Cloud node and Edge nodes with various computing resources.
  • Deploy Docker containers with trusted bioinformatics workflows running HPC jobs.
