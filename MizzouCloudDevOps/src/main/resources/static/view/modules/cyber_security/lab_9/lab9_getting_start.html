Purpose of the Lab

The purpose of the lab is to help deploy machine learning models on cloud platforms in a secure way, ensuring that both the model and data are protected from external access, while also maintaining the integrity of computations when the data is in use. This is achieved using Confidential Computing — a technology that enables data to be processed in a protected environment, also known as a Trusted Execution Environment (TEE). These environments isolate sensitive data and code from the rest of the system, even from cloud providers or malicious insiders, ensuring that computations remain private and unaltered.

Goals/Outcomes
  • Configuring Certifier Framework: simulating an enclave within a Virtual Machine.
  • Single-Client Deployment: allowing client to establish a connection with the server.
  • Scaled-Up Deployment: allowing multiple clients to establish connections with the server.
