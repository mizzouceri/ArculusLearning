Purpose of the Lab

In this chapter, we will explore Ansible as an automation tool for managing servers in the cloud. You will learn how to use AWS cloud to bring up a simple Ansible topology that has 3 nodes - one Control node and 2 Managed nodes. Then you will go through several modules to learn how Ansible can be used to automate processes, e.g., user management and software/library management. In the last module in this lab, you will go through a use case application designed using cyber defense basics to learn about how to use Ansible to manage different hosts in the AWS cloud, including a vulnerable server and an attacker, and then you will perform some attack/defense practices in these two hosts.

Goals/Outcomes
  • Understand Ansible concepts and commands.
  • Deploy Ansible web application playbooks.
  • Deploy cybersecurity operation use case application.
