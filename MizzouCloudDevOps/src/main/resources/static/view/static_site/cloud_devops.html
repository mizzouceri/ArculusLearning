About Mizzou Cloud and DevOps

Mizzou Cloud and DevOps is a learning platform that consists of an educational catalog of hands-on laboratory exercises based on various technologies, platforms, and real applications. The web portal provides a user-friendly interface with lab instruction manuals. Students can use the Mizzou Cloud and DevOps to exercise implementation of trending technologies such as Containerization, CI/CD through DevOps, Monitoring and management of infrastructure/application. The lab exercises are built using standardized tools that are open for the industry. It provides flexibility to set, reset, customize, scale and extend virtualized and admin-managed Cloud and DevOps modules.
