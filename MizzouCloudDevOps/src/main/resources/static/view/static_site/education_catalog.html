Educational Catalog


Learning Modules

Mizzou Cloud and DevOps labs are designed to allow students to use real-life applications to experience emerging technologies such as Cloud and DevOps. Students are tasked to create containers, use DevOps tools, monitoring and management tools.

Time to Complete - 3 hours
Module 1: Kubernetes with Visual Cloud Computing app

This Lab is a basic implementation of streaming and analyzing video data and object tracking algorithms. Learners will gain Kubernetes skills by using NRP Nautilus cloud platform. Hands-on skills on Kubernetes fundamentals such as Pod, Deployment, Service, Load Balancing, Ingress, Persistent Volumes. This module can be performed on PRP Nautilus or AWS cloud platforms.

Time to Complete - 3 hours
Module 2: Software Defined Networking with Cyber Defense app

This lab teaches SDN capabilities using VXLAN protocol. Learners will learn basics of how to create virtual network overlay on GCP and create a virtual network topology for performing defense by pretense using Dolus system. Hands-on skills to be learning include various tools such as: Frenetic, Tshark, Scapy, Docker, MySql, Shell, Python, and OVS vSwitch. This module can be performed on GCP or AWS cloud platforms.

Time to Complete - 3 hours
Module 3: Edge Computing with KubeEdge

Learners explore KubeEdge for edge computing with trusted bioinformatics. This module show how a KubeEdge cluster can orchestrate high performance computing workflows on selected nodes within a cluster. Hands on experience on: KubeEdge, Edge Computing. This module can be performed on Open Cloud (GENI) or AWS cloud platforms.

Time to Complete - 3 hours
Module 4: CI/CD with Jenkins with data-intensive web app

This module helps in understanding CI/CD using Jenkins tool. The lab uses an app called KnowCOVID-19 which will be used as a basis for showing continuous integration and continuous delivery. Hands on experience on: Jenkins, App development, CI/CD. This module can be performed on AWS cloud platform.

Time to Complete - 3 hours
Module 5: VMWare Tanzu for Kubernetes Cluster Management

This module helps in management of Kubernetes cluster using the VMWare Tanzu Community Edition. The lab uses a Drone Video Processing Pipeline example to demonstrate the basics and utility of the Tanzu platform.

Time to Complete - 3 hours
Module 6: Ansible for Infrastructure as Code with Cybersecurity Ops app

This module is built to teach Ansible and its concepts to the learners for provisioning, configuration management, and application deployment.

Time to Complete - 2 hours
Module 7: Kubeflow on AWS with Smart Scheduling App

In this lab module you will learn how to deploy Machine Learning pipelines using Kubeflow on AWS with an appointment scheduling use case application.

Time to Complete - 2 hours
Module 8: Prometheus and Grafana on AWS with Video Transmission App

In this lab module, you will learn how to deploy and configure Prometheus and Grafana for monitoring the metrics of a real time video transmission application.

Time to Complete - 3 hours
Module 9: Confidential Computing to Secure Data Processing Using Certifier Framework

In this lab module, you will learn how to configure the Certifier Framework for simulating a Confidential Computing environment to execute bioinformatics workflows.
