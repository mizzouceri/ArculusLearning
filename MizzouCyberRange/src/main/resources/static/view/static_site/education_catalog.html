Educational Catalog


Learning Modules

Each lab is designed to allow students to learn about cyber security flaws in real-life applications. Student are tasked to analyze security flaws and fix them using necessary network and data science tools.

Time to Complete - 2.5 hours
Module 1: Kubernetes on PRP Nautilus with Visual Cloud Computing App

This Lab is a basic implementation of streaming and analyzing video data and object tracking algorithms. Learners will gain Kubernetes skills by using PRP Nautilus cloud platform. Hands-on skills on Kubernetes fundamentals such as Pod, Deployment, Service, Load Balancing, Ingress, Persistent Volumes.

Time to Complete - 3 hours
Module 2: Software Defined Networking on GCP with Cyber Defense App

This lab teaches SDN capabilities using VXLAN protocol. Learners will learn basics of how to create virtual network overlay on GCP and create a virtual network topology for performing defense by pretense using Dolus system.Hands-on skills to be learning include various tools such as: Frenetic, Tshark, Scapy, Docker, MySql, Shell, Python, and OVS vSwitch.

Time to Complete - 3 hours
Module 3: Edge Computing with KubeEdge

Learners explore KubeEdge for edge computing with trusted bioinformatics. This module show how a KubeEdge cluster can orchestrate high performance computing workflows on selected nodes within a cluster. Hands on experience on: KubeEdge, Edge Computing.

Time to Complete - 4 hours
Module 4: CI/CD with Jenkins on AWS with data-intensive web app

This module helps in understanding CI/CD using Jenkins tool. The lab uses an app called KnowCOVID-19 which will be used as a basis for showing continuous integration and continuous delivery. Hands on experience on: Jenkins,App development, CI/CD.
