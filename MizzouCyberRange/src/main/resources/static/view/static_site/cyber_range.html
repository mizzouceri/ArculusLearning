About Mizzou Cloud DevOps

Mizzou Cloud DevOps is a cyber defense learning platform that consists of an educational catalog of immersive, hands-on laboratory exercises divided on the basis of difficulty levels. It aims for easy-to-deploy cyber defense practice scenarios on a cloud platform by implementing a user-friendly interface with the capability to launch infrastructures with all the required tools and libraries installed. Students can use the Mizzou Cloud DevOps to launch, detect and defend against an cyber attacks in a safe environments. The lab exercises are built using standardized tools that are open for the industry. It provides flexibility to set, reset, customize, scale and extend virtualized and admin managed cybersecurity modules.
